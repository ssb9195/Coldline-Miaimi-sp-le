# Coldline-Miaimi
Tā ir vienkārša šūterspēle, kurā spēlētājs vada nekustīgu ieroci ekrāna apakšā. No augšas krīt sarkanie mērķi, un spēlētājam tie jānošauj, pirms tie sasniedz apakšu un samazina spēlētāja veselību. Reizēm parādās zilā bonusa bumba, kas aktivizē īslaicīgu dubultu punktu reizinātāju. Spēle ir interesanta ar dinamisku, reakcijā balstītu spēles procesu, kas nepārtraukti uztur spēlētāju spriedzē. 

Mēs izmantojām bibliotēku "pygame" lai izveidotu spēli un arī "randint" moduli no "random" lai veidot mērķus ar nejaušām lielumiem.

Nākotnē, iespējams, mēs varēsim izgatavot jauna veida mērķus.
